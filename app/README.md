# Discadora Tauros

Aplicacao web PHP da Discadora Tauros.

Este diretório e instalado pelo script principal do repositório:

```bash
sudo bash install_discadora_tauros.sh
```

O instalador cria o primeiro administrador com o e-mail e senha informados durante a instalação.

Em produção, configure pelo painel:

- servidor Asterisk;
- AMI;
- MagnusBilling;
- domínio público;
- tema, logo e segurança.
